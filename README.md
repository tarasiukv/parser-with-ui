# Scraper #
Laravel parser with Admin UI.

## Link for scraper ##
- default - http://lifehacker.com/rss

## Installation ##

- Install all dependencies:
``npm install``


- Install composer:
``composer install``


- Run parse:
  - schedule (every 5 min):
``php artisan schedule:run``
  - immediately:
``php artisan parse:post``


- Build project:
``npm run build``


- Run UI:
``php artisan serve``
